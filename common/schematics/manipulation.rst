========================================
Manipulation
========================================

Rotate
----------------------------------------
``F3`` or pick handle |is-handle|

Mirror Horizontally
----------------------------------------
``F4`` or |is-mirror|

Flip Vertically
----------------------------------------
``F5`` or |is-flip|

Array
----------------------------------------
#. Select components
#. |is-array|
#. Set rows and column
#. Set spacing

Duplicate
----------------------------------------
a. Select part; then ``Ctrl+C``, ``Ctrl+V``, or
b. Select part, then ``Ctrl+LClick`` and drag ``LClick``
