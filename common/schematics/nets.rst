========================================
Nets
========================================

Add a net
----------------------------------------
#. ``n`` or |is-net|
#. ``LClick`` in start
#. Draw and ``LClick`` on end

.. only:: pads_std_plus

    Add Multiple
    ----------------------------------------
    #. |is-multinet|
    #. ``drag LClick`` to select starting pins
    #. ``drag LClick`` to destination nets
    #. ``RClick``

Add nets to pins
----------------------------------------
#. n or |is-net|
#. ``Alt+LClick`` on pin
