========================================
Parts Placement
========================================

Disperse
----------------------------------------
Tools > Disperse Components

Move components
----------------------------------------
#. Pick part
#. ``Ctrl+E`` or ``LClick and drag``

Rotate
----------------------------------------
``Ctrl+R`` or ``tab``

Change layer
----------------------------------------
``Rclick`` > Flip Side or ``Ctrl+F``

Cluster components
----------------------------------------
#. ``RCLick`` > Select Components
#. Select parts
#. ``RClick`` > Create Cluster

Select clusters
----------------------------------------
``RClick`` > Select clusters

Place parts from a cluster
----------------------------------------
#. ``LClick`` to Select Cluster
#. ``RClick`` > Break
#. Place parts
