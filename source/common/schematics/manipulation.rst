========================================
Manipulation
========================================

Rotate
----------------------------------------
.. only:: pads_std_plus

    ``F3`` or pick handle |is-handle|

.. only:: pads_maker

    ``r`` or pick handle |is-handle|

Mirror Horizontally
----------------------------------------
.. only:: pads_std_plus

    ``F4`` or |is-mirror|

.. only:: pads_maker

    ``h`` or ``m`` or |is-mirror|

Flip Vertically
----------------------------------------
.. only:: pads_std_plus

    ``F5`` or |is-flip|

.. only:: pads_maker

    ``f`` or ``v`` or |is-flip|

Array
----------------------------------------
#. Select components
#. |is-array|
#. Set rows and column
#. Set spacing

Duplicate
----------------------------------------
a. Select part; then ``Ctrl+C``, ``Ctrl+V``, or
b. Select part, then ``Ctrl+LClick`` and drag ``LClick``
