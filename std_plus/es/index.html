Cross-probing¶

It works for integrated projects.

Setup¶

On Schematics¶

  1. Setup > Settings
  2. At Cross Probing section, Check all checkboxes. Click OK.
  3. Setup > Cross Probing needs to be checked.

On Layout and Router ¶

  1. Tools > Options
  2. In Cross Probing Tab, Check “Enable”
  3. Choose Action.

Consejo

Pan is good for placing/routing

Using¶

Select in Schematics, work in Layout/Router¶

  1. Arrange Schematics and Layout/Router windows in splitted screen or different displays
  2. In Layout/Router, RClick and choose the selection type
  3. In Schematics, select the components
  4. Go to Layout/Router, in a region without elements to select press Ctrl+LClick, to avoid losing the selection
  5. Perform the selected operation

Select in Layout/Router, work in Schematics¶

  1. In Layout/Router, select the elements
  2. In Schematics, in a region without elements to select press Ctrl+LClick, to avoid losing the selection
  3. Perform the selected operation.

Vias¶

Nota

  • Pad stack La forma de la vía en las capas
  • Via Span La vía predeterminada para pasar de una capa a otra
  • Through Via Vía que atraviesa toda las capas
  • Partial Via Vía que atraviesa sólo algunas capas (vías blind y buried)

Setup¶

Set via Pad Stacks¶

  1. En Layout () Setup > Pad Stacks
  2. Pad Stack type a Via
  3. Selecciona vía y cambia propiedades

Agregar un vía padstack¶

  1. In “Pad Stacks”, defina tipo de Vía, luego Click en “Add Via”
  2. Elegir “through” o “partial”
  3. Define capa de inicio y fin para partial vias
  4. Define estilos de pads for pad y thermal

Define vías predeterminados¶

  1. En Layout () Setup > Via Spans
  2. Elige Default via para cada celda según los tipo de vías y capas

Elige vías por Capa/Clase¶

En Constraint manager () > Net Classes, set Via Assignments (el mismo diálogo de “Via Spans”)

Definitions¶

Net Classes¶

To set Via assignments, layers where to route, trace widths and diff pair spacing. Applicable to nets.

Clearances¶

To set general clearances (via to via, via to pad, etc). Just use the (Default Rule), except if custom clearances between Net classes is required (for example, distance between two buses).

Constraint classes¶

To set stub lengths, topology, net lengths, max number of vias, and diff pair separation distance

Descargar partes¶

Descargar una sola parte¶

  1. Abrir PADS Maker Schematic y cargar un proyecto
  2. En los checkboxes, activar “Symbol/Footprint”
  3. En la barra de búsqueda, ingresar el nombre de la parte
  4. (Recomendado) activa también “RoHS Complaint” y “Lead Free”
  5. Click en “More” en la fila seleccionada
  6. Revisar que no hayan advertencias en la parte
  7. Click en “Download”. Seleccionar la partición si es necesario
  8. Una vez que la parte se descargó (archivo .pqz), hacer click sobre ella
  9. En PADS Maker Schematics, la parte será automáticamente incluida en la biblioteca “PartQuestDirect”

Consejo

Puedes usar filtros para reducir el número de resultados

Bajar múltiples partes¶

  1. Sigue los mismos pasos que para una sola parte, pero en vez de hacer click en “Download’m click en “Add to Project”. Selecciona el proyecto o crea uno nuevo.
  2. Cuando todas las partes han sido incluidas, hacer click a la izquierda en “My Projects” y seleccionar en proyecto
  3. Hacer click en las partes a descargar o en “Select All”
  4. Click en “Download” sobre los encabezados de la tabla.
  5. Doble click en el archivo .pqz para agregar las nuevas partes a la biblioteca.
